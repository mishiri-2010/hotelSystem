# hotelSystem
This is hotel management system typically designed for administrative use only. It is used to manage the room details there availability, price room number and type.
admin has a mandate to create, add, edit and delete room details.

HOW TO INSTALL DEPENDENCES
1. dowload and install composers
2. initialise the project using terminal.
3. install xampp in your device make sure Apache and my SQL are running
4. ensure the folder htdocs have all our system files

   the you are good to run hotel system management
